# TicketFind

TicketFind is a React app built with Vite.

## Scripts

Install dependencies:

```bash
npm install
```

Production build (output goes to `docs/` for GitHub Pages):

```bash
npm run build
```

Local development:

```bash
npm run dev
```

## GitHub Pages

Pages serves the static files from `docs/` after `npm run build`; it does not run Vite on the server.
